## Project Overview
A personalized movie recommendation system built using PYTHON and MACHINE LEARNING. <br/>
-  Suggests movies to users based on **their preferences**.
-  Demonstrates skills in **data preprocessing, similarity algorithms, and model evaluation**.

## Features
- User-friendly **input/output interface** (notebook). <br/>
- Implements **content based filtering**.<br/>
- Generates **Top 30 recommended movies** with titles by processing similar genres.
- Includes recommendations.

## Technologies Used
